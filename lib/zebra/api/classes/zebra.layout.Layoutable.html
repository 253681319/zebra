        Class zebra.layout.Layoutable

        Class defined in: layout.js:107

        Layoutable class defines rectangular component that has elementary metrical properties like width, height and location and can be a participant of layout management process. Layoutable component is container that can contains other layoutable component as its children. The children
        components are ordered by applying a layout manager of its parent component.

          path expression.
